Steps to install:

1. Open VS Code terminal or cmd or powershell
2. Paste this: git clone https://github.com/addiiillll/MotoSphere.git
3. Go to Backend using this command: cd backend
4. Run the code: "npm i" then run "npm run dev"
5. Go to Frontend using this command: cd frontend
6. Run the code: "npm i" then run "npm run dev"
7. Make sure to add .env file in each Backend and Frontend.
